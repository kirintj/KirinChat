# CSS 与布局知识点

## 盒模型
- 标准盒模型 vs IE 盒模型
- box-sizing: border-box 的作用
- margin 合并现象

## 布局方案
- Flexbox：主轴/交叉轴、flex-grow/shrink/basis
- Grid：二维布局、grid-template、grid-area
- 定位：static/relative/absolute/fixed/sticky
- 浮动（历史方案，了解即可）

## BFC（块级格式化上下文）
- 触发条件：overflow/float/position/display
- 作用：清除浮动、阻止 margin 合并

## 响应式设计
- 媒体查询 @media
- 相对单位：rem/em/vw/vh
- 移动优先策略

## 动画与过渡
- transition：过渡效果
- animation + @keyframes：复杂动画
- transform：变换（translate/rotate/scale）
- will-change 性能优化
