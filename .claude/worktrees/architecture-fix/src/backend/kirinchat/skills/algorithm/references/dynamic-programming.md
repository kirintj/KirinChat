# 动态规划知识点

## 核心思想
- 最优子结构：大问题的最优解包含子问题的最优解
- 重叠子问题：同一子问题被多次计算
- 状态转移方程：定义子问题之间的关系

## 解题步骤
1. 定义状态（dp 数组含义）
2. 写出状态转移方程
3. 确定初始条件
4. 确定遍历顺序
5. 推导验证

## 常见类型
- 背包问题：0-1 背包、完全背包、多重背包
- 子序列问题：最长递增子序列(LIS)、最长公共子序列(LCS)
- 区间 DP：矩阵链乘法、石子合并
- 树形 DP：树的最大独立集
- 状态压缩 DP：位运算表示集合

## 优化技巧
- 滚动数组：空间优化
- 前缀和：区间求和优化
- 单调队列：滑动窗口优化
- 记忆化搜索：自顶向下备忘录
