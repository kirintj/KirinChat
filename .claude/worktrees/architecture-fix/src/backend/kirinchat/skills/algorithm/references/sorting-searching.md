# 排序与搜索知识点

## 排序算法
- 冒泡排序：O(n²)，稳定
- 选择排序：O(n²)，不稳定
- 插入排序：O(n²)，稳定，小数据量高效
- 归并排序：O(n log n)，稳定，需要额外空间
- 快速排序：O(n log n) 平均，不稳定，原地排序
- 堆排序：O(n log n)，不稳定，原地排序
- 计数/桶/基数排序：线性时间，适用特定场景

## 搜索算法
- 二分查找：O(log n)，要求有序数组
- 二分变体：查找左边界/右边界/第一个满足条件的位置
- BFS：层序遍历，最短路径（无权图）
- DFS：深度遍历，回溯

## 双指针
- 快慢指针：链表环检测、中点查找
- 对撞指针：两数之和、接雨水
- 滑动窗口：最长子串、最小覆盖子串

## 技巧
- 排序 + 双指针
- 哈希表空间换时间
- 二分答案（搜索解空间）
