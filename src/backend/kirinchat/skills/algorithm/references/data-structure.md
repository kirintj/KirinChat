# 数据结构知识点

## 数组与链表
- 数组：连续内存、O(1) 随机访问、O(n) 插入删除
- 链表：非连续内存、O(1) 插入删除、O(n) 查找
- 双向链表、循环链表、跳表

## 栈与队列
- 栈：LIFO，用于函数调用栈、表达式求值、括号匹配
- 队列：FIFO，用于 BFS、任务调度
- 单调栈/单调队列：解决滑动窗口最值、下一个更大元素

## 哈希表
- 哈希函数、冲突解决（链地址法/开放寻址法）
- 负载因子与扩容
- HashMap vs TreeMap 选择

## 树
- 二叉树遍历：前序/中序/后序/层序
- BST：插入/删除/查找 O(log n)
- AVL 树、红黑树（自平衡）
- 堆：最大堆/最小堆、优先队列
- 字典树（Trie）：字符串前缀匹配
- 线段树、树状数组：区间查询

## 图
- 邻接矩阵 vs 邻接表
- 有向图/无向图/加权图
- 连通分量、拓扑排序
