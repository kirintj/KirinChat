# 图论知识点

## 图的表示
- 邻接矩阵：空间 O(n²)，查边 O(1)
- 邻接表：空间 O(n+e)，查边 O(degree)

## 遍历算法
- BFS：队列实现，层序遍历，最短路径（无权）
- DFS：递归/栈实现，路径搜索，连通分量

## 最短路径
- Dijkstra：非负权图，O((V+E)log V)
- Bellman-Ford：可处理负权，O(VE)
- Floyd-Warshall：全源最短路，O(V³)

## 拓扑排序
- 适用于 DAG（有向无环图）
- Kahn 算法（BFS + 入度）
- DFS 后序翻转
- 应用：任务调度、课程安排

## 并查集
- 基本操作：Find / Union
- 路径压缩 + 按秩合并
- 应用：连通分量、Kruskal 最小生成树

## 最小生成树
- Prim：贪心扩展，适合稠密图
- Kruskal：边排序 + 并查集，适合稀疏图
