# 数据库设计知识点

## 范式化设计
- 1NF：每列原子性
- 2NF：消除部分依赖
- 3NF：消除传递依赖
- BCNF：更严格的 3NF

## 反范式化
- 适当冗余减少 JOIN
- 宽表提升查询性能
- 适合读多写少场景

## 分库分表
- 垂直分库：按业务拆分
- 水平分表：按 ID 取模/范围分片
- 分片中间件：ShardingSphere/Vitess

## 读写分离
- 主库写、从库读
- 主从复制延迟问题
- 强制读主库的场景

## 索引设计
- 联合索引最左前缀
- 覆盖索引避免回表
- 索引下推优化
- 避免索引失效的场景

## SQL 优化
- EXPLAIN 分析执行计划
- 慢查询日志分析
- 避免 SELECT *
- 合理使用 LIMIT
