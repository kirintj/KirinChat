组件实现规范

数值基线锁定
每个组件的尺寸、间距、圆角必须与参考模板的 numeric_baseline 一一对应。不允许近似值。
五态交互层
hover / pressed / focus / active / select 使用 ::before/::after 伪元素实现，颜色引用 --harmony-interactive-* token。
液态玻璃材质
浮层组件（titlebar/bottomtab/dialog/drawer）使用 backdrop-filter: blur() saturate() + mix-blend-mode: plus-lighter。
响应式适配
桌面端 ≥768px：侧边栏导航 + 宽屏布局
移动端 <768px：BottomTab + 360px 画布
