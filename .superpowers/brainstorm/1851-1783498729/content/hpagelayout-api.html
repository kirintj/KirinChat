HPageLayout — 组件 API 设计

Mode-based 布局组件，5 种模式覆盖所有页面类型

Props

mode: 'sidebar' | 'list' | 'editor' | 'chat' | 'center' // 布局模式（必填）
sidebarWidth: number = 280 // sidebar 模式下的侧边栏宽度（px）
sidebarCollapsed: boolean = false // 侧边栏是否折叠
padded: boolean = true // 内容区是否添加内边距（chat 模式自动 false）
scrollable: boolean = true // 内容区是否可滚动
Events
update:sidebarCollapsed(value: boolean) // 侧边栏折叠状态变化
